- Root LICENSE missing.
- Package/project metadata license mismatch.

- Transitive dependency licenses not reviewed where tooling allows.
- Strong copyleft obligations unidentified.
- Weak copyleft obligations unidentified.
- Unknown/custom/no-license packages.
- Dev-only versus distributed runtime license risk not distinguished.
- Fonts without source/license.
- Icons without source/license.
- Images/audio/video without source/license.
- Templates/example data without source/license.
- Generated code provenance unclear.
- Models, weights, embeddings, or datasets without license review.

- Modification notice obligations ignored.
